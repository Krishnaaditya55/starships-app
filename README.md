
# Star wars Starships Statistics App

A basic React app for Star Wars Data


## Run Locally

Clone the project

```bash
  https://github.com/Krishnaaditya55/starships-app
```

Go to the project directory

```bash
  cd  starships-app
```

Install dependencies

```bash
  npm install
```

Start the App

```bash
  npm start
```


## Technologies

- ReactJS

- Bootstrap

- Sass

- Animate.css
